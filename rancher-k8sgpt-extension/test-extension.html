✅ Rancher Extension Bundle

This is a JavaScript module that Rancher loads dynamically.

What you're seeing:

  • Minified JavaScript code (normal for production builds)
  • UMD format (Universal Module Definition) - works in browsers, Node, AMD
  • All Vue components bundled together
  • Rancher shell components externalized (loaded from Rancher itself)

How it works:

  1. Rancher loads this .js file via Developer Load
  2. Rancher executes the code in its own context
  3. The code registers UI components with Rancher's plugin system
  4. You see "K8sGPT" appear in Rancher's left sidebar
  5. Clicking it shows your list/detail views inside Rancher

This is NOT a standalone web page

You don't view this file directly. Rancher loads and executes it.

To actually use it:

1. Open Rancher UI 2. Go to: ☰ → Local → Extensions 3. Click: ⋮ → Enable Extension Developer Features 4. Click: Developer Load 5. Enter: http://127.0.0.1:4500/k8sgpt-1.0.0/k8sgpt-1.0.0.umd.min.js 6. Click: Load

Then "K8sGPT" will appear in Rancher's sidebar!

What's in the bundle:

  • ✅ Product registration (sidebar item)
  • ✅ List view component (ResourceTable)
  • ✅ Detail view component (tabs, diagnostics)
  • ✅ SeverityBadge component
  • ✅ Model class (stateColor, stateDisplay, etc.)
  • ✅ CSS styles (scoped)
